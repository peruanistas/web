# Peruanistas

Web project for Peruanistas, the citizen participation platform.

![The Peruanistas feed: "El damero"](./static/design.png)

```txt
src/components: Reusable application components
src/styles: Reusable application styles (css)
src/pages: The application page routes
```

## Getting Started

```sh
# Install dependencies
npm install
# Run the development server
npm run dev
```
